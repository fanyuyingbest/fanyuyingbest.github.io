# 工作铁律

  • 不谈论领导，不评价同事，不谈论自己(前车之鉴)
  • 不说不该说的话，不做不该做的事，专注自己的工作
  • 不好管闲事，不好为人师，有问题会来主动找你
  • 不把生活中的情绪带到工作中，工作中不能参杂其他情绪
